APILab Desktop App is a cross-platform desktop application that helps you create and manage API requests. It is built on top of the [APILab Web Client](/documentation/clients/web) and is powered by [Tauri](https://tauri.app).

## Install APILab Desktop App

1. Download the latest version of APILab Desktop App from the links above or from [official website](https://hoppscotch.com/download).
2. Open the downloaded file.
3. Follow the on-screen instructions to install APILab Desktop App.
4. Open APILab Desktop App.
5. If you see a warning message, click "**Open**".

## Access APILab

### APILab Cloud Edition for Individuals

Seamlessly access APILab Cloud Edition from APILab Desktop App:

1. Open APILab Desktop App.
2. Click the APILab logo in the top-left corner.
3. Click "**HOPPSCOTCH CLOUD**".
4. Sign in with your APILab Cloud account to access your workspaces and collections.

### APILab Self-Hosted Edition for Community

<Warning>
  To enable desktop app support for your self-hosted APILab instance, make sure to update the `WHITELISTED_ORIGINS` environment variable in your `.env` file with your deployment URL.
</Warning>
<Note>
  e.g. to allow connection to `https://hoppscotch.my-domain.com` you need to add `app://hoppscotch_my_domain_com` and `http://app.hoppscotch_my_domain_com` to the `WHITELISTED_ORIGINS` environment variable.
  ```bash
  WHITELISTED_ORIGINS=...existing_origins,app://hoppscotch_my_domain_com,http://app.hoppscotch_my_domain_com
  ```
  <Tip>
    _app://hoppscotch_my_domain_com_ &nbsp; for Linux and macOS machines.
  </Tip>
  <Tip>
    _http://app.hoppscotch_my_domain_com_ &nbsp; for Windows machine.
  </Tip>
</Note>

Add your self-hosted APILab Community Edition instance to APILab Desktop App:

1. Open APILab Desktop App.
2. Click the APILab logo in the top-left corner.
3. Click "**Add an instance**".
4. Enter the URL of your self-hosted APILab instance.
5. Click "**Connect**".

<Tip>You can also self-host APILab Desktop App. Follow the instructions in the [APILab GitHub repository](https://github.com/hoppscotch/hoppscotch/tree/main/packages/hoppscotch-desktop).</Tip>
