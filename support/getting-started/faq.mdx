## General

<AccordionGroup>
  <Accordion title="What is API Lab?">
    API Lab is a free and open-source API development environment that helps you to quickly test and debug APIs. It is a web-based alternative to Postman, Swagger, and OpenAPI.
  </Accordion>
  <Accordion title="What is the difference between API Lab and Postman/Insomnia/Paw/Httpie?">
    API Lab is a web-based alternative to Postman, Insomnia, and Paw. It is a free and open-source API development environment that helps you to quickly test and debug APIs.
  </Accordion>
</AccordionGroup>

## How-to

<AccordionGroup>
  <Accordion title="How to use API Lab?">
    You can use API Lab by visiting [s.dpway.io/luongquochai0707-app-apilab](https://s.dpway.io/apilab) and start making API requests. You can learn more about API Lab by reading our [Getting started](/guides/getting-started/introduction) guide.
  </Accordion>
  <Accordion title="How to install API Lab?">
    You can install API Lab on your desktop by visiting [s.dpway.io/luongquochai0707-app-apilab](https://s.dpway.io/apilab) and clicking on the "**Install +**" button. You can also install API Lab on your mobile device by visiting [s.dpway.io/luongquochai0707-app-apilab](https://s.dpway.io/apilab) and clicking on the "**Install +**" button.
  </Accordion>
  <Accordion title="How to use API Lab on mobile?">
    We're deprecating mobile support for API Lab. You can still use API Lab on mobile by visiting [s.dpway.io/luongquochai0707-app-apilab](https://s.dpway.io/apilab) and clicking on the "**Install +**" button.
  </Accordion>
</AccordionGroup>

## Features

<AccordionGroup>
  <Accordion title="What are the features of API Lab?">
    API Lab has a lot of features that you can use to make your API development experience better. You can learn more about API Lab features by reading our [features](/documentation/features) page.
  </Accordion>
  <Accordion title="How does Collections work?">
    Collections are a way to organize your API requests. You can create a collection by clicking on the "Collections" button on the left sidebar. You can learn more about Collections by reading our [Collections guide](/documentation/features/collections).
  </Accordion>
  <Accordion title="How does Environments work?">
    Environments are a way to store variables that you can use in your API requests. You can create an environment by clicking on the "Environments" button on the left sidebar. You can learn more about Environments by reading our [Environments guide](/documentation/features/environments).
  </Accordion>
</AccordionGroup>

## Bugs

<AccordionGroup>
  <Accordion title="How to report a bug?">
    You can report a bug by clicking on the "Report a bug" button on the left sidebar. You can also report a bug by visiting our [GitHub repository](https://github.com/luongquochai/apilab/issues/new/choose).
  </Accordion>
  <Accordion title="How to report a security vulnerability?">
    You can report a security vulnerability by visiting our [GitHub repository](https://github.com/luongquochai/apilab/security/policy).
  </Accordion>
  <Accordion title="Localhost is not working">
    If you're using API Lab on your local machine, you might need to enable `CORS`. You can learn more about `CORS` by reading our [Set up guide](/documentation/getting-started/setup#locally-served-apis).
  </Accordion>
</AccordionGroup>
