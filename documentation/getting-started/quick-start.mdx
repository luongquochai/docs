## Platforms

### API Lab Cloud

API Lab Cloud is our hosted API development and testing platform that allows you to share your APIs with your team with ease. No need to worry about hosting, scaling, and maintenance. API Lab Cloud is built for individuals and teams.

[Get started for free on API Lab Cloud](https://app.apilab.work)

### API Lab Self-Host

For full data ownership and control over your API development and testing, you can self-host API Lab on your infrastructure.

<CardGroup cols={2}>
  <Card title="Community Edition" iconType="light" icon="circle-arrow-right" href="/documentation/self-host/community-edition/getting-started">
    API Lab Community Edition is free and open-source. It is licensed under the MIT License. You can use it for personal and commercial projects. It is a great choice for individuals and small teams.
  </Card>
  <Card title="Enterprise Edition" iconType="light" icon="circle-arrow-right" href="/documentation/self-host/enterprise-edition/getting-started">
    SAML-based SSO, on-prem deployment, audit logs, and more. API Lab Enterprise Edition is a self-hosted version of API Lab Cloud with enterprise-ready features for teams and organizations.
  </Card>
</CardGroup>

<Tip>API Lab Enterprise Edition is available for on-premise deployment with priority support. [Contact API Lab Support](/support/getting-started/contact)</Tip>

## Solutions

### API Lab Web App

The easiest way to get started with API Lab is with our hosted cloud plan. Get unlimited collections with no limits and all of our features are built for individuals and teams.

[Open API Lab Web App](https://app.apilab.work)

### API Lab Desktop App

API Lab Desktop App is a cross-platform desktop application built with Tauri and API Lab Web Client. It is a standalone version of API Lab that can be installed on your computer and used without a browser. API Lab Desktop App is available for Mac, Windows and Linux.

[Download API Lab Desktop App](https://hoppscotch.com/download)

### API Lab CLI

API Lab CLI is a command-line tool that allows you to run API Lab on your terminal and CI/CD pipelines.

[Install API Lab CLI](/documentation/clients/cli/overview)
